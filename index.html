div 태그 영역입니다

HTML 태그의 종류는 다음과 같습니다

  • html 웹 페이지의 시작과 끝을 알리는 태그
  • head 브라우조 화면에는 직접 보이지 않고 숨은 데이터들을 정의하는 태그가 들어감
  • body 문서의 몸통이자 화면에 보이는 것들이 주로 들어감
  • title 웹 페이지의 제목을 나타나는 태그
  • meta head안에 보통 들어가서 웹페이지에는 보이지 않는 정보를 제공하는데 쓰이는 태그
  • div 영역을 뜻하거나 구획을 나눌 때 사용
  • a anchor 웹 페이지나 외부사이트를 연결하는 태그
  • script 실행 가능한 코드를 웹 페이지에 넣기위해 사용
  • link 해당 문서와 외부 소스(external resource) 사이의 관계를 정의할 때 사용
  • img 이미지로 정의되면 문서에 직접 넣어지는게 아니고 html 문서에 이미지가 링크되는 형태
